# ThaiGlow LITE v12 - Iris Shader

A lightweight shader pack for Minecraft with Iris.

## Installation

1. Download this shader pack
2. Extract to your Minecraft `shaderpacks` folder
3. Reload shaders or restart Minecraft
4. Go to Options → Shader Options → Configure Shaders

## Settings

All settings are available under:
- **LIGHTING**: Day/Night Brightness, Torch Colors
- **SHADOW**: Directional Light, Shadow Strength & Quality
- **FOG**: Fog Enable & Density
- **EFFECTS**: Sun/Moon, Clouds
- **PERFORMANCE**: Wind, Water Waves, Specular

## Features

✓ Dynamic day/night cycle
✓ Directional shadows with PCF
✓ Torch & ambient lighting
✓ Distance fog
✓ Customizable settings
✓ Works with Iris & OptiFine
